# Financial Trend Analysis Using SEC EDGAR (10-K Filings)

##  Project Overview

This project performs financial trend analysis for three major companies by extracting data from SEC EDGAR 10-K filings. The analysis focuses on key financial metrics over the last three fiscal years and evaluates performance using Python and Pandas. The data is cleaned, structured, analyzed, and visualized to identify year-over-year growth and overall financial trends.

##  Objectives

* Extract financial data from SEC EDGAR 10-K filings
* Analyze key financial metrics for the last three fiscal years
* Perform year-over-year (YoY) growth calculations
* Conduct aggregate and trend analysis
* Visualize financial trends using graphs
* Summarize insights and conclusions

##  Companies Analyzed

##  Financial Metrics Used

##  Project Workflow

1. Data Extraction

   * Retrieved financial data from SEC EDGAR 10-K filings
   * Extracted last three fiscal years for each company

2. Data Cleaning & Preparation

   * Structured data into a Pandas DataFrame
   * Converted data into pivot format
   * Handled missing values using mean/interpolation

3. Data Storage

   * Exported cleaned data to Excel format
   * Converted Excel to CSV for DataFrame analysis

4. Data Analysis

   * Calculated Year-over-Year growth for each metric
   * Performed aggregate analysis by company and year
   * Conducted trend analysis

5. Data Visualization

   * Created line charts for financial trends
   * Compared performance across companies

6. Insights & Conclusions

   * Evaluated growth patterns
   * Compared profitability and asset expansion
   * Identified financial stability and volatility

##  Key Analysis Performed

* Year-over-Year Growth Calculation
* Aggregate Analysis by Company
* Aggregate Analysis by Year
* Multi-level Grouping (Company & Year)
* Trend Analysis
* Graphical Visualization

##  Results Summary

* Microsoft demonstrated steady and consistent growth
* Apple showed strong operating cash flow and profitability
* Tesla exhibited higher growth potential with more volatility
* All companies showed expansion in total assets
* Liabilities increased moderately across companies

##  Conclusion

The analysis indicates that all three companies maintain strong financial performance. Microsoft reflects stability, Apple demonstrates strong cash flow management, and Tesla shows aggressive growth with higher variability. Overall, the financial trends highlight positive growth across the analyzed period.
